# Task Manager API
## 📝 Descrição
O TaskManager API é o motor de um sistema de produtividade pessoal. Este projeto foi construído para demonstrar o domínio de tecnologias modernas de backend, focando em segurança, escalabilidade e organização de código.

## 🛠️ Tecnologias Utilizadas
Framework: .NET 8 (Web API) 
Linguagem: C# 
Banco de Dados: MySQL 
ORM: Entity Framework Core 
Autenticação: JWT (JSON Web Token) 
Documentação: Swagger (OpenAPI) 

## 🏗️ Arquitetura
O projeto segue o padrão de camadas para garantir a separação de responsabilidades:

Controllers: Exposição dos endpoints REST.
Services: Regras de negócio da aplicação.
Repositories: Abstração do acesso a dados.
Models/DTOs: Definição das entidades e objetos de transferência.
